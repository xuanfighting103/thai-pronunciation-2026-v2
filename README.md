# 泰語聽辨教室 v3

給台灣學習者使用的泰語清濁音與聲調聽辨平台，可直接部署到 GitHub Pages。

## 本版改善

- 真人審核音檔優先，泰語 TTS 僅作為備援
- 移除改變 pitch 的男女聲模擬，避免扭曲聲調
- 只列出 `th-TH` 泰語語音，並優先裝置內建語音
- 只需受試者編號（例如 A01），首次輸入時自動建立
- 已加入 47 個真人錄音詞，可對應 52 道題目
- 獨立記錄登入次數、最近登入、作答時間與音檔來源
- 題組正確率與弱項報告
- 目前學生 CSV 匯出，以及全系統 JSON 備份／還原
- 手機、平板與鍵盤操作優化

## 加入正確發音

本版已將提供的 47 個 M4A 真人音檔放入 `assets/audio`，並在 `data/audio-manifest.json` 建立 52 道題目的對應。之後若要增加錄音，請閱讀 `assets/audio/README.md`。

## 重要限制

這仍是純靜態網站。受試者編號及學習紀錄只存在目前瀏覽器，沒有密碼驗證。跨裝置同步與研究者即時集中查看需要雲端後端（例如 Supabase 或 Firebase）與適當的資料權限設計。

## 本機預覽

不要直接雙擊 `index.html`，因為瀏覽器會阻擋 JSON 載入。請在專案目錄啟動任何靜態伺服器，例如 `python3 -m http.server 8080`，再開啟 `http://localhost:8080`。
